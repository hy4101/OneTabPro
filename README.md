项目名称：OneTab Pro

OneTab Pro更实用的新一代标签页管理工具

更友好的管理界面

更友好的使用体验

支持数据云同步

## 项目运行

1、安装依赖：npm install

2、启动项目：npm run dev-onetab-pro

3、项目打包：npm run build-onetab-pro
